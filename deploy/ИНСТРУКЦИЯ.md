# Деплой КОНДА на Timeweb VPS

## Шаг 1 — Создать сервер на Timeweb

1. Войдите в [timeweb.cloud](https://timeweb.cloud)
2. **Облачные серверы** → **Создать сервер**
3. Настройки:
   - **ОС**: Ubuntu 22.04 LTS
   - **Тариф**: минимальный (1 CPU, 1 GB RAM достаточно)
   - **Имя**: `conda-server` (любое)
4. Нажмите **Создать**
5. Дождитесь запуска (~2 минуты) — вам придёт email с **IP-адресом**, **логином** (root) и **паролем**

---

## Шаг 2 — Подключиться к серверу по SSH

**На Windows** — откройте PowerShell или cmd и выполните:
```
ssh root@ВАШ_IP_АДРЕС
```
Введите пароль из письма.

**Или** используйте [PuTTY](https://putty.org/) — введите IP, порт 22, логин root.

---

## Шаг 3 — Настроить DNS домена

В панели вашего **регистратора домена** (где покупали домен):

Добавьте A-записи:
| Тип | Имя | Значение |
|-----|-----|---------|
| A   | @   | ВАШ_IP_АДРЕС |
| A   | www | ВАШ_IP_АДРЕС |

> DNS распространяется до 24 часов, обычно 15–30 минут.

---

## Шаг 4 — Запустить скрипт установки

На сервере (в SSH-сессии) выполните по одной строке:

```bash
# 1. Скачать скрипт
curl -O https://raw.githubusercontent.com/ekbkondadom-ctrl/Proekt_Managers/main/deploy/setup.sh

# 2. Открыть скрипт и вписать ваш домен (первая строка после комментариев)
nano setup.sh
```

В редакторе **nano** найдите строку:
```
DOMAIN="ВАШ_ДОМЕН.ru"
```
Замените `ВАШ_ДОМЕН.ru` на ваш реальный домен, например `konda.ru`.

Сохраните: **Ctrl+O** → Enter → **Ctrl+X**.

```bash
# 3. Запустить установку
bash setup.sh
```

Скрипт сам установит всё необходимое (~5 минут). В конце спросит про SSL.

---

## Шаг 5 — Первый вход

Откройте в браузере `http://ВАШ_ДОМЕН.ru`

При первом открытии сайт попросит создать аккаунт **Супер-администратора** — придумайте логин и пароль.

---

## Частые команды (на сервере)

```bash
pm2 status                    # Статус приложения
pm2 logs conda-backend        # Логи в реальном времени
pm2 restart conda-backend     # Перезапустить приложение
bash /var/www/conda/deploy/update.sh   # Обновить до последней версии
```

---

## Обновление сайта после изменений

Когда будете вносить правки в код и хотите обновить сайт:

1. На своём компьютере: `git push` (как обычно)
2. На сервере: `bash /var/www/conda/deploy/update.sh`

---

## Если что-то пошло не так

```bash
pm2 logs conda-backend --lines 50   # Посмотреть ошибки
nginx -t                            # Проверить конфиг nginx
systemctl status nginx              # Статус nginx
```
